## The panel swap (both families)
Old "Vegetation response" panel (n=1 plot scatter, x = within-year wet-extent sqrt) → **community cloud + unit marker** (x = between-year flood frequency, y = veg_p05 floor, GAM-cloud form, C1 checkerboard hues). The marker is a **census aggregate over the unit's own pixels** (mean; median on the QA sidecar), reconciling exactly to the polygon `100·Σwet/Σvalid`. Every panel carries the three honesty/support lines. Paddock panels additionally state the dominant community's **area share** ("shown on Inland Floodplain (wet) — 73% of this paddock").
