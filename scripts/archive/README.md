# Scripts Archive

provenance only. Do not use these as active workflow entry points unless a
future task explicitly promotes them.

The controlling manifest is `docs/scripts_manifest.csv`. Stage 3 moved legacy
`scripts/obs/` files into `obs_legacy_20260627/` and moved one-off utilities
into `one_off_utilities_20260627/`.

Current workflow scripts live in the numbered role folders under `scripts/`.
